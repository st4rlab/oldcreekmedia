Installation

Get up and running in less than 10 minutes.

Install miniml's source and JavaScript files via clone, fork, or npm. As simple or as complex as you like.

Download

Quick Install

The easiest way to install quickly is to clone the GitHub repository.

git clone https://github.com/st4rlab/miniml.git

Make sure you have jekyll installed and you can run a build and live test with this:

bundle install
bundle exec jekyll serve

Or install it yourself as:

$ gem install miniml
